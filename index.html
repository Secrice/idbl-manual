날짜별 기록

날짜별 통계를 확인하고 에버리지·하이런 그래프를 조회합니다.

  • 플레이어 옆 핸디(다마 수)는 그날 적용한 평균 핸디를 나타냅니다.
  • 경기목록 링크를 클릭하면 세부 경기 데이터를 확인할 수 있습니다.
  • 에버리지를 클릭하면 그날의 에버리지 그래프가 표시됩니다.
  • 하이런을 클릭하면 그날의 하이런 그래프가 표시됩니다.

그룹 선택 후 날짜별 기록을 보면 해당 월의 통계만 표시됩니다. 테이블 헤더를 클릭하여 정렬 순서를 변경할 수 있습니다.

경기 진행 방법

게임 정보 입력

경기 시작 전 선수 정보, 핸디, Game ID를 설정합니다.

플레이어 설정

선수1에는 선공 선수를 입력합니다. 흰 공이 기본 선공이며, 색⇅ 버튼으로 공 색상을 변경하고, 순서 버튼으로 선·후공을 바꿀 수 있습니다.

점수에는 핸디(다마 수)를 직접 입력합니다. 100, 200 단위가 아닌 개수(예: 17, 18)로 입력합니다.

iDBL은 100·200 구간으로 나뉘는 4구가 아닌, 1개 단위로 세분화된 핸디를 추구합니다.

경기 진행 방법

에버리지 기반 자동 핸디

에버Now 값을 기준으로 핸디를 자동 계산하여 승률 50%를 목표로 합니다.

에버리지 체크박스를 활성화하면 현재 에버Now를 기준으로 핸디가 자동 설정됩니다. 에버Now는 최근 20경기 중 최고·최저 에버리지를 제외한 평균값입니다.

핵심 공식

4구 기준 — 에버리지 1 = 15개 핸디. 즉 에버리지 × 15 를 기본 핸디로 계산합니다.

핸디 범위 적용 단위 처리 방식 예시
1 ~ 49개 1개 단위 그대로 적용 에버 2.0 → 30개
50 ~ 99개 5개 단위 5 단위 올림 에버 3.6 → 54개 → 55개
100개 이상 10개 단위 10 단위 올림 에버 7.0 → 105개 → 110개

에버리지 체크를 해제하면 핸디를 수동으로 직접 입력할 수 있습니다.

경기 진행 방법

Game ID

진행 중인 경기를 식별하는 고유 번호입니다.

Game ID는 전체 게임에 대한 고유 값이 아니라 현재 진행 중인 게임 내에서의 식별자입니다. 경기 종료 후 약 3일이 지나면 자동 삭제됩니다. 자동 생성된 값을 그대로 사용하면 됩니다.

경기 재개 방법

전화 수신이나 앱 비정상 종료로 경기가 중단된 경우, 동일한 Game ID를 입력하고 이어서 버튼을 누르면 기존 게임을 이어서 진행할 수 있습니다. Game ID는 메인 페이지 진행 경기 목록에서 확인 가능합니다.

1
게임 시작

자동 생성된 Game ID를 확인합니다.

2
경기 중단 발생

전화, 앱 종료 등 예기치 않은 중단이 발생합니다.

3
Game ID 입력 후 이어서

진행 경기 목록에서 Game ID를 확인하고 입력한 뒤 이어서 버튼을 누릅니다.

조작 방법

  • 👆
    스코어보드 윗쪽을 클릭하면 +1점, 아랫쪽을 클릭하면 -1점입니다.
  • 🔄
    다른 선수의 점수 영역을 클릭하면 턴이 전환됩니다.
  • ↩️
    턴 뒤로 버튼을 누르면 이전 턴으로 되돌아갑니다. 단, 현재 턴에서 득점이 없어야 합니다.
  • 🏆
    점수가 파란색(3쿠션 마무리)으로 변하면 +1을 한 번 더 누릅니다. "OO가 승리하였습니까?" 팝업에서 확인을 눌러야 결과가 기록됩니다.
  • 🆕
    새 게임 버튼을 누르면 현재 게임을 취소하고 새 경기를 시작합니다.

경기 중간에 멈추는 것은 결과에 기록되지 않습니다. 승패를 기록하려면 반드시 승리 팝업에서 확인을 눌러주세요.
